# Abbreviations

This document provides a reference for abbreviations used throughout the Physical AI & Humanoid Robotics book.

## A

**AI**: Artificial Intelligence
**ARIA**: Autonomous Robot Interaction Agent (the example robot name used in this book)
**API**: Application Programming Interface
**AR**: Augmented Reality
**ASCII**: American Standard Code for Information Interchange

## C

**CAD**: Computer-Aided Design
**CPU**: Central Processing Unit
**CNN**: Convolutional Neural Network
**CSP**: Constraint Satisfaction Problem

## D

**DOF**: Degrees of Freedom
**DMP**: Dynamic Movement Primitives
**DL**: Deep Learning
**DP**: Dynamic Programming

## E

**EKF**: Extended Kalman Filter
**EMG**: Electromyography
**ETH**: Eidgenössische Technische Hochschule (Swiss Federal Institute of Technology)

## F

**FPGA**: Field-Programmable Gate Array
**FIR**: Finite Impulse Response

## G

**GPU**: Graphics Processing Unit
**GCD**: Greatest Common Divisor
**GUI**: Graphical User Interface
**GCS**: Ground Control Station

## H

**HRI**: Human-Robot Interaction
**HUD**: Head-Up Display
**HMM**: Hidden Markov Model
**HTML**: HyperText Markup Language
**HTTP**: HyperText Transfer Protocol

## I

**IMU**: Inertial Measurement Unit
**IR**: Infrared
**IoT**: Internet of Things
**IP**: Internet Protocol
**IDE**: Integrated Development Environment

## J

**JSON**: JavaScript Object Notation
**Jacobian**: In vector calculus, a matrix of all first-order partial derivatives of a vector-valued function

## K

**KPI**: Key Performance Indicator
**KUKA**: A German manufacturer of industrial and service robots
**Kalman**: Named after Rudolf E. Kalman, a mathematician who developed the Kalman filter

## L

**LiDAR**: Light Detection and Ranging
**LID**: Liquid Interface Detection
**LSTM**: Long Short-Term Memory
**LAN**: Local Area Network

## M

**ML**: Machine Learning
**MIT**: Massachusetts Institute of Technology
**MPC**: Model Predictive Control
**MIMO**: Multiple Input Multiple Output
**MVC**: Model-View-Controller

## N

**NLP**: Natural Language Processing
**NVIDIA**: A technology company known for GPUs and AI computing platforms
**NIST**: National Institute of Standards and Technology

## P

**PID**: Proportional-Integral-Derivative (controller)
**PDF**: Probability Density Function or Portable Document Format
**PC**: Personal Computer
**P2P**: Peer-to-Peer
**PCL**: Point Cloud Library

## R

**ROS**: Robot Operating System
**ROS 2**: Robot Operating System 2
**RL**: Reinforcement Learning
**RNN**: Recurrent Neural Network
**RGB**: Red Green Blue
**RGB-D**: Red Green Blue with Depth information
**RTK**: Real-Time Kinematics

## S

**SLAM**: Simultaneous Localization and Mapping
**SVM**: Support Vector Machine
**SDK**: Software Development Kit
**SD**: Secure Digital (memory card)
**SISO**: Single Input Single Output
**SNN**: Spiking Neural Network

## T

**TTS**: Text-to-Speech
**ToF**: Time of Flight
**TCP**: Transmission Control Protocol
**TLS**: Transport Layer Security
**UART**: Universal Asynchronous Receiver-Transmitter

## U

**UI**: User Interface
**UX**: User Experience
**USB**: Universal Serial Bus
**UDP**: User Datagram Protocol
**UKF**: Unscented Kalman Filter

## V

**VR**: Virtual Reality
**VSLAM**: Visual Simultaneous Localization and Mapping
**VNC**: Virtual Network Computing
**VPN**: Virtual Private Network

## W

**WiFi**: Wireless Fidelity
**WLAN**: Wireless Local Area Network
**WPILib**: A library for programming robots in FIRST Robotics Competition
**WebRTC**: Web Real-Time Communication

## X

**XML**: eXtensible Markup Language
**XHTML**: eXtensible HyperText Markup Language

## Y

**YAML**: YAML Ain't Markup Language

## Z

**ZMP**: Zero Moment Point
**ZED**: A type of stereo camera made by StereoLabs

## Greek Letters

**μ (mu)**: Often represents friction coefficient, mean, or micro prefix
**σ (sigma)**: Often represents standard deviation or conductivity
**θ (theta)**: Often represents an angle, particularly joint angles in robotics
**ω (omega)**: Often represents angular velocity
**λ (lambda)**: Often represents eigenvalues, wavelength, or Lagrange multipliers
**π (pi)**: Mathematical constant (approximately 3.14159...)
**τ (tau)**: Often represents torque or time constants
**φ (phi)**: Often represents angles, particularly in spherical coordinates
**ψ (psi)**: Often represents the yaw angle in 3D space
**ρ (rho)**: Often represents density or correlation coefficient
